Error Propagation

  • Notes
    • All the results are stored locally, meaning that all data will be lost if the site data for this webpage is cleared.
    • Use respective Ans (e.g., "Ans1", "Ans2", etc.) as the average to use previously calculated values for better precision
    • Use "×/÷" to calculate reciprocals. Put the number on the first line, choose "÷", remove the second line, and hit "Calculate"
    • [Experimental] To calculate SigFigs, use appropriate SigFigs in the input (e.g., "3.0" instead of "3")
  • Keyboard shortcuts
    • Shift+Enter: Run calculation
    • q: Add row (when applicable)
    • w: Remove row (when applicable)
    • s: Change "Extra SigFigs - Required SigFigs" switch
    • d: Change "SigFig rules - SigFig rules + 2" switch
    • z: Remove last result, if any
    • r: Change mode to "+/-"
    • t: Change mode to "×/÷"
    • y: Change mode to "ln"
    • u: Change mode to "log"
    • i: Change mode to "e^"
    • o: Change mode to "10^"
    • p: Change mode to "a^x"
    • All the inputs, including the dropdown, can be navigated to using Tab and Shift+Tab. Use Enter to open the dropdown, ↑ and ↓ to select functions, and Enter to submit choice
  • Data formats accepted
    • Numbers: 3.1415926; -2020; .57721
    • Scientific notations: 8e8; .897e-12; 599e+3
    • Previous answers: Ans1; Ans22
    • All empty inputs are treated as 0.
